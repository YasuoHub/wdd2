# TabBar 图标说明

## 需要的图标文件（共8个）

| 图标 | 文件名 | 状态 | 建议颜色 |
|------|--------|------|---------|
| 首页 | `home.png` | 未选中 | #999999 |
| 首页 | `home-active.png` | 选中 | #07c160 |
| 大厅 | `hall.png` | 未选中 | #999999 |
| 大厅 | `hall-active.png` | 选中 | #07c160 |
| 消息 | `message.png` | 未选中 | #999999 |
| 消息 | `message-active.png` | 选中 | #07c160 |
| 我的 | `my.png` | 未选中 | #999999 |
| 我的 | `my-active.png` | 选中 | #07c160 |

## 图标规格

- **尺寸**: 81 x 81 像素（实际显示 27 x 27pt）
- **格式**: PNG
- **背景**: 透明
- **风格**: 线性图标（未选中）、填充图标（选中）

## 推荐获取方式

### 方式1：阿里巴巴图标库（推荐）
访问 [iconfont.cn](https://www.iconfont.cn)，搜索以下关键词下载：
- 首页: "home"、"首页"
- 大厅: "list"、"分类"、"任务"
- 消息: "message"、"消息"、"聊天"
- 我的: "user"、"我的"、"个人中心"

### 方式2：使用 Emoji 风格图标
可以使用以下图标风格参考：
- 🏠 首页
- 📋 大厅
- 💬 消息
- 👤 我的

### 方式3：简单文字图标
如果不想用图片，可以临时使用文字标签（需要在 app.json 中配置）

## 当前状态

目前这些文件是**占位符**，你需要替换为实际的图标文件。
如果没有图标文件，小程序 TabBar 会显示空白。

## 快速替换脚本（使用 iconfont 图标）

```bash
# 下载图标后，重命名为以下格式：
# home.png, home-active.png
# hall.png, hall-active.png
# message.png, message-active.png
# my.png, my-active.png

# 将图标放入此目录即可
```
