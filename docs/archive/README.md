## 当前产品口径

1. 现金/余额悬赏，不再是积分悬赏。
2. 积分仅作为平台权益体系，不参与任务支付、退款、结算或提现。
3. 帮助者通过任务大厅主动接单，当前没有自动派单或人工派单。
4. 消息置顶、消息删除、会话归档暂不做。
5. 发布前必须清理真实用户路径中的模拟任务、测试用户和模拟评分。
