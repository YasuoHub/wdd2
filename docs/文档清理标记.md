## 当前产品口径

1. 当前发布前版本是“现金/余额悬赏 + 钱包结算”的真实交易产品，不再是“积分悬赏”产品。
2. 积分只用于平台权益体系，不可提现、不可转让、无现金价值，也不参与任务支付、退款、结算。
3. 帮助者通过任务大厅主动接单，当前没有派单功能。
4. 消息置顶、消息删除、会话归档暂不做，不属于发布前阻塞项。
5. 发布前必须清理所有会在真实用户路径中展示的模拟任务、测试用户、模拟评分、模拟列表。

## 当前暂不做的功能

| 功能 | 当前状态 | 原因 |
| --- | --- | --- |
| 消息置顶 | 暂不做 | 不影响发布、接单、聊天、结算核心闭环 |
| 消息删除 | 暂不做 | 涉及聊天证据和争议处理，早期不建议允许用户随意删除 |
| 会话归档 | 暂不做 | 真实消息量不足前没有必要 |
| 自动派单 | 暂不做 | 当前先验证任务大厅主动接单是否成立 |
| 人工派单 | 暂不做 | 会增加运营成本和责任边界，后续需要单独设计 |
