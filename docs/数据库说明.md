# 问当地数据库说明

**更新日期**：2026-05-14  
**数据库类型**：微信云数据库，文档型数据库  
**集合命名约定**：业务集合统一使用 `wdd-` 前缀

## 1. 设计原则

1. 用户身份以微信 `openid` 为可信来源。
2. 金额字段以“元”为单位存储，展示时统一格式化。
3. 任务状态、订单状态、工单状态必须使用固定枚举。
4. 资金变动必须写入记录集合，方便追踪和对账。
5. 举报、申诉、工单和任务状态强关联，但保留独立记录，方便客服回溯。

| 集合 | 用途 |

| `wdd-users` | 用户资料、余额、积分、信用、评价统计 |
| `wdd-needs` | 求助任务主表 |
| `wdd-need-takers` | 接单记录 |
| `wdd-messages` | 聊天消息 |
| `wdd-notifications` | 系统通知 |
| `wdd-payment-orders` | 微信支付订单 |
| `wdd-balance-records` | 余额流水、任务收入、提现、退款类记录 |
| `wdd-point-records` | 积分流水 |
| `wdd-sign-in-records` | 签到记录 |
| `wdd-invite-records` | 邀请记录 |
| `wdd-ratings` | 任务评价 |
| `wdd-reports` | 举报记录 |
| `wdd-appeals` | 申诉记录 |
| `wdd-tickets` | 客服工单 |
| `wdd-config` | 平台配置 |

## 3. 用户集合：`wdd-users`

用于保存用户基础资料、钱包、积分和信誉。

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| `_id` | String | 用户文档 ID |
| `openid` | String | 微信用户唯一标识 |
| `nickname` | String | 昵称 |
| `avatar` | String | 头像地址 |
| `available_points` | Number | 可用积分 |
| `frozen_points` | Number | 冻结积分 |
| `total_points` | Number | 累计积分 |
| `balance` | Number | 可提现/可用余额 |
| `frozen_balance` | Number | 冻结余额 |
| `total_earned` | Number | 累计收入 |
| `total_withdrawn` | Number | 累计提现 |
| `total_paid` | Number | 累计支付 |
| `credit_score` | Number | 信用分，默认 100 |
| `ban_status` | Object/null | 封禁状态，正常为 `null` |
| `rating` | Number | 平均评分 |
| `rating_count` | Number | 评价数量 |
| `frequent_locations` | Array | 常去地点 |
| `help_types` | Array | 愿意帮助的类型 |
| `help_willingness` | String | 帮助意愿 |
| `create_time` | Date | 创建时间 |
| `update_time` | Date | 更新时间 |

建议索引：

| 索引 | 用途 |
| --- | --- |
| `openid` 唯一索引 | 登录查询 |
| `credit_score` | 风控筛选 |

## 4. 任务集合：`wdd-needs`

用于保存求助任务主数据。

| 字段 | 类型 | 说明 |
| --- | --- | --- |
| `_id` | String | 任务 ID |
| `user_id` | String | 求助者用户 ID |
| `user_nickname` | String | 求助者昵称快照 |
| `user_avatar` | String | 求助者头像快照 |
| `type` | String | 求助类型 |
| `type_name` | String | 求助类型中文名 |
| `description` | String | 问题描述 |
| `images` | Array | 任务图片 |
| `location` | Object | GeoJSON 点位，`coordinates` 为 `[longitude, latitude]` |
| `location_name` | String | 地点名称 |
| `address` | String | 地址补充 |
| `reward_amount` | Number | 悬赏金额 |
| `platform_fee` | Number | 平台服务费 |
| `taker_income` | Number | 帮助者预计/实际收入 |
| `payment_status` | String | 支付状态 |
| `payment_order_id` | String | 关联支付订单 |
| `status` | String | 任务状态 |
| `taker_id` | String | 当前接单者 ID |
| `expire_time` | Date | 任务有效期截止时间 |
| `completed_time` | Date | 完成时间 |
| `cancelled_time` | Date | 取消时间 |
| `was_reported` | Boolean | 是否曾发起举报 |
| `was_appealed` | Boolean | 是否曾发起申诉 |
| `create_time` | Date | 创建时间 |
| `update_time` | Date | 更新时间 |

任务状态枚举：

| 值 | 中文 | 说明 |
| --- | --- | --- |
| `pending` | 待匹配 | 已支付并发布，等待接单 |
| `ongoing` | 进行中 | 已接单，双方处理中 |
| `completed` | 已完成 | 正常完成或裁定完成 |
| `cancelled` | 已取消 | 取消、退款或裁定取消 |
| `breaking` | 审核中 | 举报/申诉后等待客服处理 |

建议索引：

| 索引 | 用途 |
| --- | --- |
| `location` 2dsphere | 附近任务查询 |
| `status + expire_time` | 任务大厅和自动取消 |
| `user_id + create_time` | 我的求助 |
| `taker_id + update_time` | 我的任务 |
| `payment_order_id` | 支付订单反查任务 |

## 14. 平台配置集合：`wdd-config`

当前建议使用固定文档：`wdd-config/platform`。

| 配置 | 说明 |
| --- | --- |
| `platform_fee_rate` | 平台服务费率 |
| `min_reward_amount` | 最低悬赏金额 |
| `min_withdraw_amount` | 最低提现金额 |
| `new_user_points` | 新用户积分 |
| `invite_points` | 邀请奖励积分 |
| `sign_in_rules` | 签到积分规则 |
| `supplement_deadline_hours` | 工单补充材料截止时长 |
| `customer_service_openids` | 客服人员 OpenID 白名单数组，用于判断客服权限 |

## 15. 权限建议

1. 普通用户只能通过云函数写入核心业务集合。
2. `wdd-users`、`wdd-needs` 可按需开放只读字段，敏感字段通过云函数返回。
3. 支付、提现、工单裁定集合必须禁止前端直接写入。
4. 图片上传建议限制文件大小、类型和目录。
